**Architecture:** Keep the existing microservice layout. Implement Academic Service and Payment Service independently with layered Spring Boot code: controller, service, repository, entity, dto, and exception handling.

**Tech Stack:** Java 21/Spring Boot 3, Spring Data JPA, Bean Validation, MySQL, Maven, React/Vite, Docker Compose.
